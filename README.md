**OSINT Intelligence Platform** — a single-file web application that aggregates open-source intelligence tools into one unified interface.

## Features

- **People Search** — Query multiple people-finder services (TruePeopleSearch, WhitePages, Spokeo, PeekYou, and more) from a single form
- **Username Enumeration** — Check username availability across social platforms with WhatsMyName integration
- **Email Lookup** — Cross-reference email addresses across breach databases, social platforms, and verification services
- **Domain Intelligence** — SSL certificate search (Censys), internet-connected device lookup (Shodan), DNS analysis, and WHOIS data
- **Multi-Source Aggregation** — Launches searches across dozens of OSINT sources simultaneously
- **Multiple Themes** — Dark, light, and hacker (green terminal) theme options
- **100% Client-Side** — Runs entirely in the browser, no server or API keys required
- **Single HTML File** — Zero dependencies, zero install, just open and use

## Usage

1. Download or clone the repository
2. Open `index.html` in any modern browser
3. Enter your search parameters and select a category
4. Results open in new tabs across multiple OSINT sources
